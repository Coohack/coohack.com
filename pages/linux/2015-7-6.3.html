好了，先上张fedora的艳照：

是不是挺美的，底部docker和应用浏览页面。下面说一下gnome的美化方法：

首先是窗口按钮放在左边：
安装dconf: dnf install dconf
进入后找到：org->gnome->desktop->wm->preferences
然后在右边找到button-layout这个选项，更改为close,minimize,maximize:appmenu
这样按钮就能显示在左边了。

然后是docker
安装tweaks: dnf install tweak
进入后左边选择extensions，右边最下面会有get more extensions，点击以后浏览器会打开安装插件页面，直接搜索dock就能找到一个simple docker了
还有一个插件需要安装，就是经用左上角的hotcorner，直接搜索 No Topleft Hot Corner 就能找到了，安装以后左上角热点就不能触发。
当然，tweak里面还需要开启user themes 插件，用来后面安装图标。

安装numix-circle图标
安装fedy: dnf install fedy
安装好以后可以执行: fedy -e Numix-Circle
之后在tweak里面的appearance中的icons选择刚刚下载好的主题就好了，这只是换了图标，窗口主题就用paper主题，
paper主题的安装要用源码，很简单的地址https://github.com/snwh/paper-gtk-theme.git，下载好以后以管理员身份运行install.sh就行了
